# Online Quiz App (Python + Tkinter)

A simple and interactive GUI-based quiz application built using Python
and Tkinter.\
This project is beginner-friendly but strong enough to showcase in
resumes and interviews.

## Features

-   Graphical User Interface (GUI) using Tkinter
-   Multiple-choice questions
-   Score calculation
-   Displays final result
-   Restart quiz option
-   Clean and simple code structure

## Technologies Used

-   Python 3
-   Tkinter (GUI library)

## Project Structure

quiz_app/ - quiz_app.py - README.md

## How to Run the Project

1.  Make sure Python is installed on your system\
    Check version: python --version

2.  Make sure Tkinter is working: python -m tkinter

3.  Run the project: python quiz_app.py

## How It Works

-   The app displays one question at a time
-   User selects an option by clicking a button
-   The app checks the answer and updates the score
-   After all questions, final score is displayed
-   User can restart the quiz
